I install the below addons on all instances of Firefox.

Available as a collection [https://addons.mozilla.org/en-US/firefox/collections/16973778/Default/](https://addons.mozilla.org/en-US/firefox/collections/16973778/Default/)


- [Copy Selection as Markdown](https://addons.mozilla.org/en-US/firefox/addon/copy-selection-as-markdown/)
	- [GitHub - 0x6b/copy-selection-as-markdown: Firefox add-on to copy selection as Markdown](https://github.com/0x6b/copy-selection-as-markdown)

- [Dark Reader](https://addons.mozilla.org/en-GB/firefox/addon/darkreader)
    - [darkreader/darkreader: Dark Reader Chrome and Firefox extension](https://github.com/darkreader/darkreader)

- [GSConnect](https://addons.mozilla.org/en-GB/firefox/addon/gsconnect/)
    - [GSConnect/gnome-shell-extension-gsconnect: KDE Connect implementation for GNOME](https://github.com/GSConnect/gnome-shell-extension-gsconnect)

- [I still don't care about cookies](https://addons.mozilla.org/en-US/firefox/addon/istilldontcareaboutcookies/)
    - [OhMyGuus/I-Dont-Care-About-Cookies: Debloated fork of the extension "I don't care about cookies"](https://github.com/OhMyGuus/I-Dont-Care-About-Cookies)

- [Merge Windows](https://addons.mozilla.org/en-GB/firefox/addon/merge-window/)
    - [GitHub - jonathanKingston/merge-windows: A Firefox extension for merging windows](https://github.com/jonathanKingston/merge-windows)

- [Privacy Redirect](https://addons.mozilla.org/en-US/firefox/addon/privacy-redirect/)
    - [SimonBrazell/privacy-redirect: A simple web extension that redirects Twitter, YouTube, Instagram & Google Maps requests to privacy friendly alternatives.](https://github.com/SimonBrazell/privacy-redirect)
 
- [SingleFile](https://addons.mozilla.org/en-US/firefox/addon/single-file/)
	- [GitHub - gildas-lormeau/SingleFile: Web Extension for Firefox/Chrome/Edge and CLI tool to save a faithful copy of an entire web page in a single HTML file](https://github.com/gildas-lormeau/SingleFile)

- [Tab Center Reborn](https://addons.mozilla.org/en-GB/firefox/addon/tabcenter-reborn/)
    - [ariasuni / Tab Center Reborn Â· GitLab](https://framagit.org/ariasuni/tabcenter-reborn)

- [uBlock Origin](https://addons.mozilla.org/en-GB/firefox/addon/ublock-origin/)
	- [GitHub - gorhill/uBlock: uBlock Origin - An efficient blocker for Chromium and Firefox. Fast and lean.](https://github.com/gorhill/uBlock)

- [Wallabagger](https://addons.mozilla.org/en-GB/firefox/addon/wallabagger)
    - [wallabag/wallabagger: Chrome / Firefox / Opera plugin for wallabag v2.](https://github.com/wallabag/wallabagger)
